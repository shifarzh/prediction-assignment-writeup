Prediction Assignment Writeup

This HTML report summarizes the process of building a machine learning model using Random Forest to predict the manner in which barbell exercises were performed.

Steps

  1. Loaded and cleaned the data
  2. Partitioned the dataset
  3. Trained a Random Forest model with cross-validation
  4. Evaluated the model with confusion matrix
  5. Predicted on 20 new test cases

Model achieved high accuracy (> 99%) and is suitable for deployment.
